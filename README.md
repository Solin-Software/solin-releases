# Solin — Free Media App for Jehovah's Witnesses Congregation Meetings

[![GitHub Releases](https://img.shields.io/github/v/release/Solin-Software/solin-releases)](https://github.com/Solin-Software/solin-releases/releases)
[![Platform](https://img.shields.io/badge/Platform-Windows-blue)](https://solinav.vercel.app/)
[![Languages](https://img.shields.io/badge/Languages-Multiple-green)](https://solinav.vercel.app/)

**Solin** is a free, lightweight, and powerful desktop application built specifically for audio and video management during Jehovah’s Witnesses congregation meetings. It automatically organizes weekly meeting media, handles songs, videos, images, PDFs, and jwpub files, and provides instant projection, Wi-Fi sharing, and excellent support for hybrid (Zoom) meetings — all with **zero setup** required.

Developed by a single dedicated brother as a service to the congregation, Solin pulls content directly from jw.org and focuses on real-world meeting flow: quick media access, clean projector output, and reliable offline operation.

- **Official Website**: [https://solinav.vercel.app/](https://solinav.vercel.app/)
- **GitHub Releases**: [https://github.com/Solin-Software/solin-releases](https://github.com/Solin-Software/solin-releases)

## ✨ Core Features

### 1. Automatic Weekly Meeting Media
- Reads the current and next week’s meeting schedule from jw.org.
- Lists every song, image, and video organized by section.
- One-tap or fully automatic download with intelligent caching — works completely offline.

### 2. Songs & Original Songs
- Full jw.org song library with search by number or title.
- Play in order, shuffle, or announce titles silently before playback.
- Offline download support.

### 3. Integrated JW.org Browser + Live Casting
- Built-in browser to navigate any jw.org content without leaving the app.
- One-click projection or live tab casting to the secondary screen/projector.
- Spotlight tool (cursor-following highlight) for presentations.

### 4. Advanced Projection
- Instant send of any media (images, videos, PDFs, jwpub) to a second screen or projector.
- Safe zoom (Ctrl + scroll / +/-) and pan (drag) directly on the projected content.
- Controls remain on the main screen; output is always clean fullscreen.
- Support for managing multiple secondary monitors individually by clicking the monitor icon in the bottom left, next to the settings button.
- Window mode display for virtual meetings.

### 5. Wi-Fi Sharing (QR Code)
- Scan a QR code with any phone or tablet to push media directly into Solin over the local network.
- No cables, no accounts, no extra apps — works instantly.

### 6. Playlists & Custom Media
- Import `.jwlplaylist` files or build playlists manually.
- Drag-and-drop reordering.
- Supports: images, videos, audio, PDFs, jwpub.

### 7. OBS Studio Integration (Hybrid Meetings Ready)
- Connects via OBS WebSocket (default port 4455).
- Automatically switches between “Camera” and “Media” scenes when you project or stop projection.
- Excellent support for hybrid meetings — media reaches remote brothers with zero manual intervention.

### 8. Memorial of Jesus’ Death
- Astronomically precise algorithm calculates the exact date every year.
- Automatically selects the official event image and can set it as the idle screen.

### 9. Idle Screen & Extras
- Set any local image or video as idle background.
- Independent UI language vs. content language configuration.
- Full offline mode and cache management.

### 10. Available Languages
The Solin interface is available in multiple languages. Media content is pulled directly from jw.org and works in **all languages supported by the official site**.

Features and screenshots are available on the [official website](https://solinav.vercel.app/features/).
